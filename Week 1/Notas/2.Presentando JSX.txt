1-JSX produce “elementos” de React. JSX es una combinacion del lenguaje XML en conjunto con javascript, por
eso podemos notar una combinacion entre lenguaje HTML y javascript 

2-JSX puede utilizar estructuras de control en conjunto con etiqutas html

3-Utilizamos llaves para  insertar expresiones JavaScript como atributos

4-JSX representa objetos:
// Nota: Esta estructura está simplificada
const element = {
  type: 'h1',
  props: {
    className: 'greeting',
    children: 'Hello, world!'
  }
};
